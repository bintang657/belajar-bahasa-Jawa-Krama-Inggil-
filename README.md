# Sinau Basa Jawa Krama Inggil

Platform pembelajaran Bahasa Jawa Krama Inggil interaktif dengan UI 3D.

## Fitur
- Kosakata Krama Inggil lengkap
- Dialog percakapan interaktif
- Aksara Jawa
- Kuis interaktif dengan skor
- UI 3D dengan animasi penuh
- Background canvas 3D bintang
- Custom cursor
- Responsive design
